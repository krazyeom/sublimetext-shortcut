Sublimetext Shortcut
====================

* PDF로 배포되어서 오타 및 버그를 수정을 못해서 직접 삽을 들어서 옮겨서 작성했습니다. 

서브라임 텍스트의 단축키 리스트입니다.

폰트는 개발자들이 가장 선호한다는 Consolas를 사용하였습니다.

유료 폰트 지만 무료로 설치하는 방법은 다음을 참조하세요.
